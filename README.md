# ISSP-Triumf-435

Setup Development Environment:

LINUX:  
        cd ISSP-Triumf-435;
        python3 -m venv venv;
        source venv/bin/activate;
        pip3 install -r requirements.txt;

WINDOWS: 
        cd ISSP-Triumf-435;
        python -m venv venv; 
        ./venv/bin/activate;
        pip install -r requirements.txt;

\
\
Start Flask app:

    flask run;
